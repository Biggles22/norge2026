# Norge2026

Statisk planeringssite för rutten Åre-Trondheim-Åndalsnäs-Geiranger med fokus på trafikkameror, vägstatus och körpunkter i Norge.

## Sidor

- `index.html` - startsida, rutter, kamerapunkter och checklista.
- `styles.css` - all styling.
- `script.js` - filtrering och rendering av kamerapunkter.
- `assets/norway-road-hero.png` - genererad hero-bild.

## Datakällor

- Statens vegvesen trafikkarta: https://www.vegvesen.no/trafikk
- Statens vegvesen webkamera: https://www.vegvesen.no/webkamera
- Nasjonale turistveger Geiranger-Trollstigen: https://www.nasjonaleturistveger.no/en/routes/geiranger--trollstigen/
- Datasetbeskrivning: https://dataut.vegvesen.no/dataset/webkamera

Vegvesens råa webkamera-publicering via DATEX kräver separat åtkomst. Därför länkar första versionen till officiella livevyer istället för att bädda in bildflöden.
